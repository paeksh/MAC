 CAS4109 Assignment #2 - Mandatory Access Control (MAC)
================================================================

----------------------------------------------------------------
Overview
----------------------------------------------------------------

CLI 도구 MAC 을 구현하는 코드
    - read-down, write-up 정책에 따라 현재 로그인된 유저가 파일에 읽거나 쓰는 동작을 수행한다
    - 리눅스의 setuid/setgid 기능을 이용한다

파일 세팅
    - mac.policy 파일에 {유저:보안등급} 형식의 문자열로 저장되어 있고, root 권한으로 읽을수 있다
    - secret.data, top_secret.data, unclassified.data, confidential.data 네 개의 파일이 제공된다

권한 세팅
    - chown root:root *.data mac.policy 
        >> chown은 소유자와 그룹을 바꾸는 명령어.
        >> 소유자:그룹 둘다 root으로 바꿈
        >> *.data : .데이터로 끝나는 모든 파일
    - chmod 0640 <files>
        >> chmod는 권한을 바꾸는 명령어.
        >> 0640은 -rw-r------을 숫자로 표현한것
        
----------------------------------------------------------------
Functions
----------------------------------------------------------------

level_t enumerate
    -  4개의 등급을 숫자로 구분한다. (등급이 매겨지지 않으면 -1)

drop_privilege()
    - if 안에서 setegid((getgid))을 호출하고 비교까지 함
    - 먼저 그룹을 권한 하락 시킨 다음에 유저 권한을 하락 시켜야함. 순서 중요

get_user_clearance()
    - 파일을 한줄씩 읽어 ":" 를 기준으로 strtok()를 이용해 자르고, username을 매칭 후 그 유저의 보안등급을 반환함
    - char *strtok(char *str, const char *delim); : 첫 호출에 str에 자를 문자열을 주고, 이후 호출부터는 NULL을 줌

do_read()
    - 파일을 읽기 모드로 열고, fread로 읽은 만큼을 fwrite로 stdout에 출력한 뒤 fclose로 닫음
    - 호출전 메인에서 권한 검사를 하므로, 혀용된 경우에만 호출됨
    - 데이터 파일은 root:root 권한이므로 권한 드롭 이전 euid가 root인 상태에서 호출되어야 접근 가능

do_write()
    - fopen을 append 모드로 열어 기존 내용을 보존하며 뒤에 추가함
    (write 모드 "w"는 기존 내용을 덮어쓰므로 사용하지 않는다.)
    - fprint로 data 다음에 개행을 붙여 기록한뒤 fclose로 닫음
    - do_read와 이하동문.

logging()
    - 사용자 이름, read/write, 파일 이름을 받아 <username>.log 파일에 (없으면 생성) <command> <filename> 형식으로 기록한다.
    - 여기서는 fopen()말고 open()을 써야한다. 왜냐하면 과제 명세서에서는 권한 0640으로 로그 파일을 접근해야 한다고 되어있는데 fopen()권한을 직접 지정할수 없기 때문이다. 
    - open(<filename>, <권한 플래그>, <권한 번호>) 
        권한 플래그의 종류
        O_WRONLY : 쓰기 전용
        O_CREAT : 없으면 생성 (생성시 세번째 인자 0640이 적용됨)
        O_APPEND : 끝에 이어쓰기
        이 플래그들을 or 로 이어붙여 쓰기모드+없으면만들기+끝에 붙이기 기능을 구현
    - dprintf 는 printf의 fd 버전. 명령과 파일명을 공백으로 잇고 끝에 개행을 붙여서 한줄을 기록함
